# RESURGO — BRAND VOICE GUIDE
> How Resurgo speaks. Use this for all copy: marketing pages, coach messages, emails, UI labels, onboarding.

---

## 1. CORE VOICE ATTRIBUTES

| Attribute | What it means | What it is NOT |
|---|---|---|
| **Direct** | One sentence. One meaning. No padding. | Verbose, hedging, over-explaining |
| **Calm** | Steady authority, never pressure | Frantic, guilt-tripping, FOMO-heavy |
| **Operator-like** | Talks to someone who ships things | Motivational-poster cheerleading |
| **Intelligent** | Assumes the user is smart | Condescending, over-simplified baby-talk |
| **Anti-hype** | Honest about what it does and doesn't do | Buzzword soup, inflated claims |
| **Grounded** | Real talk, no toxic positivity | Hollow affirmations |

## 2. VOICE IN ACTION

### Headlines
✅ "Drop your goal. Get a real plan in 90 seconds."  
✅ "Stop planning. Start executing."  
✅ "One command center. Every system that matters."  
❌ "Transform your life TODAY with the #1 AI productivity app!"  
❌ "Unlock your unlimited potential with cutting-edge AI!"

### Body copy
✅ "Resurgo breaks your goal into habits and tasks — then reminds you, coaches you, and tracks what actually sticks."  
✅ "Free plan. No card needed. Real features, not a trial."  
❌ "Our revolutionary AI-powered platform leverages advanced algorithms to supercharge your productivity journey!"

### Error messages / Empty states
✅ "No tasks yet. Add one and get moving."  
✅ "No habits tracked today. Start small — one is enough."  
❌ "Oops! Looks like you don't have any tasks yet! 😊"

### Coach tone (in-product AI messages)
✅ "You've got 3 overdue tasks. That's a signal, not a failure. Let's clean them up — pick one right now."  
✅ "Energy is low today. Let's protect your one must-do and defer the rest."  
❌ "You're doing AMAZING! Keep up the great work, champ! 🎉"  
❌ "I notice you haven't completed your tasks. Perhaps you should try harder next time?"

## 3. WORD PREFERENCES

| Use | Avoid |
|---|---|
| Operator | User, customer, client |
| Execute / Ship / Build | Work on, do, accomplish |
| System | App, tool, platform (use sparingly) |
| Coach | Assistant, chatbot, AI helper |
| Momentum | Progress (unless precision needed) |
| Brain dump | Input, enter notes |
| Check-in | Update, log |
| Armory | Dashboard (internal label) |

---

## 4. FORMATTING RULES

- **Labels:** ALL CAPS terminal-style (e.g., `TODAY'S TOP 3 TASKS`)
- **CTAs:** UPPERCASE with underscores (e.g., `START_FREE`, `OPEN_ARMORY`)
- **Body text:** Sentence case, no unnecessary all-caps
- **Emoji:** Never in formal copy. Sparingly in coach messages if emotionally warranted.
- **Ellipsis:** Avoid in headlines. OK in loading states only.
- **Exclamation marks:** Maximum 1 per page in casual copy. Zero in B2B/professional contexts.

## 6. MARKETING PAGE TONE MAP

| Page | Tone | Key job |
|---|---|---|
| Homepage / landing | Confident, punchy | Grab attention, communicate value fast |
| Features | Informative, direct | Specific claims, no fluff |
| Pricing | Transparent, calm | Remove friction, no dark patterns |
| FAQ | Helpful, conversational | Answer real questions honestly |
| Compare pages | Factual, fair | We win on relevant metrics; acknowledge competitor strengths |
| Blog | Educational, direct | Deliver real value, not SEO filler |
| Onboarding | Warm, focused | Get the user to first win |
| Coach messages | Personal, tactical | Cause useful action in the user |

---

## 7. WHAT RESURGO IS NOT

Never position Resurgo as:
- A meditation app (though it tracks wellness)
- A journaling app (though it has scratch notes)
- A team collaboration tool
- A time management app (it's an execution OS)
- A life coach replacement (coaches are AI tools, not therapists)
- "The best productivity app" without specifics

Always position Resurgo as:
- Built for solo operators who ship things
- A complete execution system (not a task manager, not just a habit tracker)
- Calm and structured, not chaotic or overwhelming
- A platform you grow into (progressive disclosure)
