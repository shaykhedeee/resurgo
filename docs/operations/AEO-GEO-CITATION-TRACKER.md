## 5) Action Rules (non-negotiable)

If citation rate for a query cluster is **< 20%**:
1. Refresh the related post with direct-answer intro.
2. Add/refresh FAQ block.
3. Add one new proof/data point.
4. Improve internal links from 2 relevant posts.
5. Re-submit URL in Search Console.

If a competitor dominates a query:
1. Create a better comparison page.
2. Add explicit "why choose Resurgo" section.
3. Add clear CTA and product evidence.
