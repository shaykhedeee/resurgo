# RESURGO — Founder Physical Execution Schedule (No AI Tasks)

> This file includes only human-executed tasks for founder/marketing team.
> Time horizon: next 8 weeks.

---

## Weekly Operating Rhythm (repeat every week)

### Monday (Strategy + Pipeline)
- [ ] Review dashboard KPIs (organic, leads, signup conversion, paid conversion).
- [ ] Pick **top 3 growth priorities** for the week.
- [ ] Assign owners + deadlines.
- [ ] Approve content topics and outreach list.

### Tuesday (Distribution + Outreach)
- [ ] Publish/ship this week’s main content asset.
- [ ] Send 20 personalized backlink outreach emails.
- [ ] Post 1 founder story thread on X + 1 LinkedIn post.
- [ ] Share in 2 founder/productivity communities (no spam).

### Wednesday (Sales + Validation)
- [ ] Do 3–5 user calls (or demos).
- [ ] Collect 2 proof points (testimonial, quote, use case).
- [ ] Update messaging based on objections heard.

### Thursday (Partnerships + PR)
- [ ] Pitch 5 newsletter placements / partnerships.
- [ ] Pitch 3 podcast opportunities.
- [ ] Follow up all outstanding outreach threads.

### Friday (Optimization)
- [ ] Review Search Console query movement.
- [ ] Review blog pages with highest impressions but low CTR.
- [ ] Update titles/meta for top 3 underperforming pages.
- [ ] Plan next week’s experiments.

## 8-Week Founder Task Plan

## Week 1 — Foundation Lock-in
- [ ] Verify GA + Search Console + Meta dashboards are collecting clean data.
- [ ] Finalize core offer messaging (one-sentence value prop).
- [ ] Build target backlink prospect sheet (first 100 domains/contacts).
- [ ] Prepare proof kit: screenshots, product demo clips, user quotes.

## Week 2 — Content + Landing Conversion
- [ ] Approve and publish missing high-intent pages/posts.
- [ ] Record short product demo video (2–3 mins).
- [ ] Update homepage hero copy and CTA hierarchy.
- [ ] Run first CTA A/B test.

## Week 3 — Backlink Sprint #1
- [ ] Send 100 manual outreach emails across 5 days.
- [ ] Secure at least 5 quality placements.
- [ ] Submit to relevant tool directories (high-quality only).

## Week 4 — Social Proof & Trust
- [ ] Collect 5 detailed testimonials.
- [ ] Add testimonials to landing/pricing pages.
- [ ] Publish one case-study style post.

## Week 5 — Community Distribution
- [ ] Ship one high-value free resource (template/checklist).
- [ ] Post in 5 niche communities with real value.
- [ ] Run one live Q&A or founder AMA.

## Week 6 — Backlink Sprint #2
- [ ] Second 100-email outreach sprint.
- [ ] Guest post on 2 relevant sites.
- [ ] Lock one newsletter sponsorship or featured slot.

## Week 7 — Conversion Hardening
- [ ] Audit full signup flow manually on desktop/mobile.
- [ ] Remove 3 friction points from onboarding.
- [ ] Improve trial-to-paid nudges (copy + timing).

## Week 8 — Scale Decision Week
- [ ] Review 8-week KPI trendlines.
- [ ] Decide top 2 channels to scale.
- [ ] Cut lowest-performing channel.
- [ ] Set next 60-day growth targets.
