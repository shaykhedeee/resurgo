## Distribution checklist per published piece

1. Publish on site
2. Repurpose to X thread
3. Repurpose to LinkedIn post
4. Post one key chart/quote as visual
5. Share in communities (value-first)
6. Add internal links to `/pricing`, `/vision-board`, `/faq`

---

## KPI checkpoints

Track weekly:

- page views
- signup conversion by page
- onboarding completion rate
- Pro upgrade rate
- Vision Board gate hit → upgrade rate

If a topic underperforms for 2 cycles, replace it.
